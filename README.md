# 労務費ダッシュボード

`対象業務.csv` と月次労務費CSVを読み込み、対象業務ごとに月別労務費と合計を表示するNetlify + Neon向けアプリです。

## 使い方

1. Neonで `neon-setup.sql` を実行します。
2. Netlifyの環境変数に `DATABASE_URL` を設定します。
3. Netlifyにこのリポジトリを接続してデプロイします。
4. 画面から `対象業務.csv` を読み込みます。
5. 画面から `2026.03.csv` のような月次CSVを読み込みます。
6. `保存` を押すとNeonに保存され、再読み込み後も復元されます。

## CSV

- 対象業務CSV: `number,name`
- 月次CSV: ファイル名は `YYYY.MM.csv`
- 月次CSVの使用列: `工番`, `建コン合計`
